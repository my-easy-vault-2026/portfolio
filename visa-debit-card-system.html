Visa Debit Card System (2026)

I built a card issuing and payment processing system from scratch, integrated with Visa for payment services. The system went live in 2025 and currently has over a thousand users and hundreds of thousands of transaction records online.

Tech Stack

The system is developed in Go, using the Gin framework for API services, GORM as the ORM tool, Redis for caching and message queuing, and MySQL as the primary database. The system is deployed on AWS cloud platform with Docker containerization.

Product Features

The product allows users to top up their virtual wallet, issue a Visa card, and make payments that directly deduct from the wallet balance, providing a seamless online and offline payment experience. The system also offers features such as peer transfers, transaction history, balance inquiry, interest and reward points, and supports multiple currencies and cross-border payments.
